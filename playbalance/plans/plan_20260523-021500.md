## 3. Traps

- **Stage 1 변경이 누적되지 않게 Stage 0 원복 필수** — 이전 cliff push + EyeBall 변경이 남아있으면 새 설계 효과 분리 불가
- **Pattern=Random 의 Count 필드는 무효** (memory: `project_spawnset_pattern_lever`) — Gear 가 Random 인지 확인 후 BaseRate 변경. ✓ Gear 는 Random (위 데이터 확인). BoyFairy 는 Circle, Ghost 는 Dense — 둘 다 Count 유효
- **StartTime/EndTime 은 Float** (오늘 발견 trap) — intValue 쓰면 silent fail. floatValue 사용
- **모델 학습 분포 외**: 후반 wall 3x 증강은 학습 환경 (Count=15/20) 에서 보지 못한 강도. 모델 행동 예측 불가, 측정 후 확인
- **PlayTrace size 함정**: per-key fetch 로 (`feedback_playtrace_size_limit`)
