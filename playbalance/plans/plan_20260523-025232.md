## 4. Traps (주의)

- **σ/μ precondition abort**: 위에서 본대로 baseline σ/μ=86% 가 stable threshold (15%) 초과. converge 가 pre-cycle abort 하면 즉시 옵션 B (수동 편집 + `/bal-run`) 로 전환.
- **enemy_damage round-trip 흡수**: 자동 trigger 시 apply_abort. 발생 시 그 knob 만 한 iter skip 하고 다음 후보 자동 시도되어야 정상. 만약 또 apply_abort_3 로 끝나면 적은 정수 자산 (3, 5 같은) 의 -15% 가 흡수된 것 — adjust_default 를 -0.30 으로 키우거나 해당 knob 만 whitelist 에서 빼는 식.
- **PlayTrace per-key fetch**: `episode.cause` 같은 sparse 키는 size 함정에 걸려 sparse 잘림. converge skill 내부 fetch 가 per-key 로 되어 있는지 산출물에서 한 번 더 확인.
- **새 knob 의 asset 매칭 첫 검증**: `early_spawn_rate` 의 `names: ["Gear", "Gear 1"]` / `late_spawn_burst_amplify` 의 6개 stem / `late_spawn_rate_amplify` 의 `ForestGuardianFairy` — 첫 본격 사용이라 stem 매칭이 silent miss 할 수 있음. iter 1 후 git diff 로 의도한 자산만 변경됐는지 확인.
